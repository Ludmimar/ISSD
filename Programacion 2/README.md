## 🗂️ Contenidos por Unidad

### 🔹 Unidad I – Fundamentos de Java
- ✅ **Sintaxis básica**: Variables, tipos de datos, operadores
- ✅ **Estructuras de control**: `if/else`, `switch`, bucles
- ✅ **Entrada/Salida**: `Scanner`, `System.out.println()`
- ✅ **Primeros programas**: Hola Mundo y ejercicios básicos

### 🔹 Unidad II – Estructuras Repetitivas
- ✅ **Bucles `for`**: Iteración clásica y for-each
- ✅ **Bucles `while`**: Condiciones y control de flujo
- ✅ **Bucles `do-while`**: Ejecución garantizada
- ✅ **Bucles anidados**: Complejidad algorítmica

### 🔹 Unidad III – Arrays y Estructuras de Datos
- 🔄 **Arrays unidimensionales**: Declaración, inicialización, acceso
- 🔄 **Arrays multidimensionales**: Matrices y arrays anidados
- 🔄 **Métodos de arrays**: Ordenamiento, búsqueda, manipulación
- 🔄 **Algoritmos con arrays**: Problemas prácticos

### 🔹 Unidad IV – Programación Orientada a Objetos
- 🔄 **Clases y objetos**: Definición, instanciación
- 🔄 **Atributos y métodos**: Encapsulación, modificadores de acceso
- 🔄 **Constructores**: Inicialización de objetos
- 🔄 **Herencia**: Jerarquías de clases, polimorfismo

### 🔹 Unidad V – Colecciones y Frameworks
- 🔄 **Listas**: ArrayList, LinkedList
- 🔄 **Mapas**: HashMap, TreeMap
- 🔄 **Sets**: HashSet, TreeSet
- 🔄 **Queues y Stacks**: Estructuras de datos especializadas

## 💻 Cómo Ejecutar los Ejercicios

### Requisitos Previos
1. **Instalar Java JDK 8+**
   ```bash
   java -version
   javac -version
   ```

2. **Instalar IntelliJ IDEA** (recomendado)
   - [Descargar IntelliJ IDEA Community](https://www.jetbrains.com/idea/download/)

### Ejecución de Ejercicios

1. **Abrir proyecto en IntelliJ IDEA**
   ```bash
   # Navegar a la carpeta del ejercicio
   cd "Programacion 2/Clase XX/EjercicioYY"
   # Abrir IntelliJ IDEA desde esta ubicación
   ```

2. **Compilar y ejecutar**
   ```bash
   # Compilar
   javac src/*.java
   
   # Ejecutar
   java -cp src NombreClase
   ```

3. **Usando IntelliJ IDEA**
   - Abrir el proyecto
   - Ejecutar con `Ctrl + Shift + F10` o botón Run
   - Ver resultados en la consola
