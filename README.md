## 📚 Descripción

Este repositorio contiene el material académico y proyectos desarrollados durante la **Tecnicatura Superior en Desarrollo de Software** del ISSD. Incluye ejercicios, proyectos, exámenes parciales y trabajos prácticos de las materias de programación.

## 🗂️ Estructura del Repositorio

### 📘 Programación 1
- **Lenguaje**: Python
- **Profesor**: Emmanuel Galiotti
- **Año**: 2025 | Semestre: 1°
- **Contenidos**: Fundamentos de programación, algoritmos, estructuras de datos, funciones

### 📗 Programación 2  
- **Lenguaje**: Java
- **Profesor**: Gabriel Arriola
- **Año**: 2025 | Semestre: 2°
- **Contenidos**: Programación orientada a objetos, colecciones, gestión de archivos

## 🚀 Tecnologías Utilizadas

- **Python 3.6+** - Programación estructurada y funcional
- **Java 8+** - Programación orientada a objetos
- **IntelliJ IDEA** - IDE para desarrollo Java
- **Eclipse** - IDE alternativo para Java

## 📋 Contenido por Materia

### Programación 1 (Python)
- ✅ 10 clases con ejercicios prácticos
- ✅ 2 exámenes parciales
- ✅ Coloquio promocional
- ✅ Proyecto de gestión de notas

### Programación 2 (Java)
- ✅ Fundamentos de Java y POO
- ✅ Estructuras repetitivas y arrays
- ✅ Ejercicios y problemas prácticos

## 🎯 Objetivos de Aprendizaje

- Dominar los fundamentos de la programación
- Desarrollar habilidades en Python y Java
- Aplicar conceptos de programación orientada a objetos
- Implementar algoritmos y estructuras de datos
- Gestionar proyectos de software

## 📖 Cómo Usar Este Repositorio

1. **Navegar por las carpetas** según la materia de interés
2. **Revisar los README específicos** de cada clase/proyecto
3. **Ejecutar los ejercicios** siguiendo las instrucciones
4. **Estudiar los ejemplos** y adaptarlos según necesidades
